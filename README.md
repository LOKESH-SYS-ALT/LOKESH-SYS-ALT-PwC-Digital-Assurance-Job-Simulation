# PwC Digital Assurance Job Simulation

## About This Project

I successfully completed the PwC Switzerland Digital Assurance Job Simulation offered by Forage. Through this simulation, I worked on real-world business scenarios involving data analytics, Power BI dashboard development, IT audit, ERP user access management, and executive reporting.

This experience helped me understand how technology, business processes, and audit controls work together in a professional environment.

## Tools Used

- Microsoft Excel

## What I Worked On

### Task 1 – Power BI Dashboard

- Imported and analyzed insurance claims and contracts data.
- Created an interactive Power BI dashboard.
- Built KPIs, charts, tables, and slicers.
- Analyzed payment amounts, claims, policies, companies, and yearly trends.

### Task 2 – ERP User Access Audit

Reviewed ERP authorization controls, including:

- Joiners
- Movers
- Leavers
- Password Policy
- User Access Management

Verified supporting documents and completed the audit testing table based on the provided evidence.

### Task 3 – Executive Presentation

Prepared a management presentation that included:

- Dashboard insights
- ERP audit findings
- Risk assessment
- Recommendations for improving user access controls
